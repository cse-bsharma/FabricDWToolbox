Fabric DW Query Performance Analyzer Help

                    
                

                

                    
What does this tool check?

                    
    
                        
  • VARCHAR(8000) - Inefficient for Delta Parquet storage
    • 
                        
  • Join Type Mismatches - Implicit conversions in join predicates
    • 
                        
  • Nested Loop Joins - Row-by-row processing (bad for OLAP)
    • 
                        
  • Many-to-Many Joins - Data explosion from non-unique keys
    • 
                        
  • Data Movement Operators - Shuffle operations for data distribution
    • 
                        
  • Missing Statistics - Tables needing FULLSCAN updates
    • 
                    

                    
                    
Authentication Methods

                    
    
                        
  • Azure AD Interactive - Opens browser for login (recommended)
    • 
                        
  • Azure AD Password - Username/password for Azure AD
    • 
                        
  • Azure AD Default - Uses Azure CLI or managed identity
    • 
                    

                    
                    
About Fabric Data Warehouse

                    
This tool is optimized for Microsoft Fabric DW which uses Delta Parquet storage. 
                    Traditional SQL Server recommendations (like nonclustered indexes) don't apply.
